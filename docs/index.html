Visão Geral

Este é um projeto desenvolvido em C++ para gerenciar os livros de uma biblioteca. O sistema oferece diversas funcionalidades, incluindo o cadastro e login de usuários, o gerenciamento de livros (adicionar, listar, alugar, editar e remover livros) e o controle dos livros alugados por cada usuário.

Funcionalidades

  • Cadastro e Login de Usuário: Cadastro de usuário com nome e senha.
  • Gerenciamento de Livros: Adicionar, listar, alugar, editar e remover livros.
  • Visualização dos Livros Alugados: Verificar e listar os livros alugados pelo usuário.

Como Rodar o Projeto

Para rodar este projeto em sua máquina, siga os passos abaixo:

  1. Pré-requisitos: Certifique-se de ter o C++ instalado em sua máquina.
  2. Compilar o código: Execute o comando `g++ main.cpp Usuario.cpp Biblioteca.cpp -o sistema_biblioteca`.
  3. Rodar o sistema: Execute o comando `./sistema_biblioteca`.

Estrutura do Código

  • Usuario.h: Define a classe `Usuario`, que armazena as informações do usuário e realiza a autenticação.
  • Usuario.cpp: Implementação dos métodos da classe `Usuario`.
  • Biblioteca.cpp: Implementação da classe `Sistema_da_Biblioteca` para o gerenciamento de livros.
  • main.cpp: Função principal que organiza a interação com o usuário e executa o sistema.
